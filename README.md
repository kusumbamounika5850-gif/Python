<h2>I. Python Fundamentals</h2>

<h3>1. Core Data Types and Variables</h3>

<p>
<strong>Data Types</strong> define the type of value a variable can store.
Python is dynamically typed, meaning the type is determined at runtime.
</p>

<pre><code>x = 10        # int
price = 9.99 # float
name = "Ram" # str
is_active = True  # bool</code></pre>

<p>
<strong>Variables</strong> are used to store data values. A variable is created
when a value is assigned using the <code>=</code> operator.
</p>

<pre><code>age = 25
city = "Delhi"</code></pre>

<p>
<strong>Type Casting</strong> converts one data type into another.
</p>

<pre><code>num = "100"
num_int = int(num)   # converts string to integer
price = float(num)  # converts string to float</code></pre>

<h3>2. Operators and Expressions</h3>

<p>
<strong>Operators</strong> are used to perform operations on values.
</p>

<pre><code>a = 10
b = 3

print(a + b)   # Addition
print(a > b)   # Comparison
print(a > 5 and b < 5)  # Logical</code></pre>
